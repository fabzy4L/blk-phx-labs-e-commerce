IMPORTANT - These are instructions to CORRECTLY edit and save label design files. Instructions with images are available here: https://help.supliful.com/en/articles/5579874-customizing-your-first-label

To edit the mockup image and label design you will need Adobe Photoshop and Adobe Illustrator programs. 

The psd design file consists of 2 parts:

(A) the product mockup image which is the first layer.
(B) the label design file which will open in ai.

To edit the mockup image and label design:

1. Open psd file
2. Click through the smart layer until you get to external ai file
3. Once the ai label design is opened you can start editing it
4. Make sure you check content and design REQUIREMENTS here:  https://help.supliful.com/en/articles/5408818-labeling-guide

To update mockup image and save label design in PDF:

5. To update the product mockup layer on ai click FILE --> SAVE (Ctrl+S or Cmd+S), go back to Photoshop smart layer (if applicable) and do the same again. Finally, go to the product mockup tab and see the result.
6. To export your mockup image click FILE --> EXPORT --> Quick export as PNG. 
7. To save your Label Design PDF file go back to the ai label design file, select all (Ctrl+A or Cmd+A) and outline text by clicking TYPE --> CREATE OUTLINES (Ctrl+O or Cmd+O). Then select all your images in the Links panel (click WINDOW --> LINKS), and choose "EMBED IMAGES" from the panel menu. Then click FILE --> SAVE AS and save it as a PDF. In the pop-up window "General" tab "Adobe PDF Preset" drop-down menu choose [High Quality Print].


Finally, you need to upload your Label Design PDF and Product Image (as a replacement of the existing image) to your Product Page. And you are done!
